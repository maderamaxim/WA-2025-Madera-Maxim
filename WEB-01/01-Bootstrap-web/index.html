MŮJ NÁZOR NA BOOTSTRAP

Myslím si že Bootstrap je vhodný na rychlé zakládání webových aplikací, u kterých není brán tak velký zřetel na vzhled, protože stránky mohou velice snadno připomínat jiné. Mně samotnému se s ním pracovalo relativně dobře, protože za mě odvede spoustu práce, které jako začínající programátor nerozumím. Je pro mě však obtížné zjistit, co všechno dokáže, protože neznám všechny utility třídy což moji práci podstatně zpomaluje. Pokud však porovnám programování vlastních stylů v jazyce CSS a jednoduchost implementovat Bootstrap prvky, je Bootstrap jasnou volbou pro tvorbu nového webu.

Tlačítka
